Данный репозиторий - это результат выполнения тестового задания от компании "ЦРТ" на вакансию "Младший инженер по тестированию (автоматизация)"

Данный репозиторий включает в себя следующие файлы:
1. test-cases.xlsx - набор тест-кейсов, направленных на тестирование формы регистрации и авторизации.
2. bugs.txt - баг репорты, найденные во время выполнения тест-кейсов
3. automation_tests.py - автоматизированные тесты, реализующие набор тест кейсов (представленных в файле test-cases.xlsx
По умолчанию все автотесты закомментированы(#) для использования определенного теста выделите часть кода и нажмите комбинацию клавиш (Ctrl + /)
